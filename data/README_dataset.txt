# Sample SONAR Dataset Structure
# This file shows the expected format of the SONAR dataset
# 
# The actual dataset should be downloaded from:
# https://www.kaggle.com/datasets/rupakroy/sonarcsv
#
# Format:
# - 60 numerical features (columns 0-59)
# - 1 target column (column 60): 'R' for Rock, 'M' for Mine
# - No header row
# - Values separated by commas
#
# Example rows:
# 0.0200,0.0371,0.0428,0.0207,0.0954,...,0.0032,R
# 0.0453,0.0523,0.0843,0.0689,0.1183,...,0.0044,M
#
# Please download the actual dataset and place it as 'sonar.csv' in this folder.
